**# 공유기, 알고 쓰면 더 좋아요**

## 1. 공유기의 기본 개념

공유기(router)는 인터넷 회선을 여러 대의 컴퓨터나 기기와 공유할 수 있도록 해주는 장치입니다. 공유기가 없으면 인터넷 회선을 사용하는 컴퓨터나 기기 각각이 인터넷 회선에 직접 연결해야 합니다. 하지만 이렇게 하면 회선의 용량이 제한되어 여러 대의 기기가 동시에 인터넷을 사용하기 어렵습니다. 또한, 컴퓨터나 기기의 위치에 따라 인터넷 속도나 연결 안정성이 떨어질 수 있습니다.

## 2. 공유기의 종류

공유기는 크게 유선 공유기와 무선 공유기로 나눌 수 있습니다. 유선 공유기는 컴퓨터나 기기를 랜선으로 직접 연결하여 인터넷을 사용하도록 해줍니다. 무선 공유기는 와이파이(Wi-Fi)를 사용하여 컴퓨터나 기기와 무선으로 연결하여 인터넷을 사용하도록 해줍니다.

## 3. 공유기의 주요 기능

공유기는 다음과 같은 주요 기능을 제공합니다.

* **인터넷 회선 공유:** 인터넷 회선을 여러 대의 컴퓨터나 기기와 공유합니다.
* **NAT(Network Address Translation):** 컴퓨터나 기기에 할당된 사설 IP 주소를 공인 IP 주소로 변환하여 인터넷을 사용하도록 합니다.
* **DHCP(Dynamic Host Configuration Protocol):** 컴퓨터나 기기에 IP 주소, 서브넷 마스크, 게이트웨이 주소 등을 자동으로 할당합니다.
* **Wi-Fi:** 와이파이를 사용하여 컴퓨터나 기기와 무선으로 연결합니다.
* **보안:** 암호화, 방화벽 등을 사용하여 인터넷 연결을 보호합니다.

## 4. 공유기 선택 시 고려할 사항

공유기를 선택할 때는 다음과 같은 사항을 고려해야 합니다.

* **인터넷 회선 속도:** 사용하는 인터넷 회선의 속도에 맞는 공유기를 선택해야 합니다.
* **연결할 기기 수:** 연결할 기기의 수에 맞는 공유기를 선택해야 합니다.
* **무선 범위:** 무선 공유기를 선택할 경우, 무선 범위가 충분한지 확인해야 합니다.
* **보안 기능:** 보안 기능이 강화된 공유기를 선택하는 것이 좋습니다.

## 5. 공유기 설치 및 설정

공유기를 설치하려면 다음과 같은 순서를 따릅니다.

1. 공유기의 전원을 켭니다.
2. 공유기와 인터넷 회선을 연결합니다.
3. 컴퓨터나 기기에서 공유기의 설정 페이지에 접속합니다.
4. 공유기의 설정을 원하는 대로 변경합니다.

공유기의 설정 페이지는 대부분 인터넷 주소 `192.168.0.1` 또는 `192.168.1.1`을 입력하여 접속할 수 있습니다.

## 6. 자주 묻는 질문

### Q. 공유기를 여러 대 연결하면 인터넷 속도가 느려지나요?

A. 네, 공유기를 여러 대 연결하면 인터넷 속도가 느려질 수 있습니다. 공유기는 한 번에 한 대의 기기와만 인터넷 연결을 할 수 있습니다. 따라서, 여러 대의 공유기가 동시에 인터넷 연결을 시도하면 속도가 저하될 수 있습니다.

### Q. 공유기의 암호를 설정하는 이유는 무엇인가요?

A. 공유기의 암호를 설정하면 외부에서 공유기에 접속하여 인터넷을 사용하지 못하도록 보호할 수 있습니다.

### Q. 공유기의 보안 설정을 강화하는 방법은 무엇인가요?

A. 공유기의 보안 설정을 강화하려면 다음과 같은 방법을 사용할 수 있습니다.

* **암호 설정:** 공유기의 로그인 암호, Wi-Fi 암호 등을 강력한 암호로 설정합니다.
* **업데이트:** 공유기의 펌웨어를 최신 버전으로 업데이트합니다.
* **방화벽 설정:** 공유기의 방화벽을 사용하여 외부에서 접근할 수 있는 포트를 제한합니다.
